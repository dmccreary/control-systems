# Chapters

This textbook is organized into 16 chapters covering 300 concepts in classical control systems.

## Chapter Overview

1. [Introduction to Control Systems](01-intro-to-control-systems/index.md) - Fundamental control concepts including feedback, open/closed-loop control, and system components.

2. [Dynamic System Properties](02-dynamic-system-properties/index.md) - Essential properties of dynamic systems including linearity, time-invariance, and LTI systems.

3. [Time-Domain Response Fundamentals](03-time-domain-response/index.md) - System responses in the time domain including natural/forced response and damping characteristics.

4. [Transient Response Specifications](04-transient-response-specs/index.md) - Quantitative measures of transient response including overshoot, settling time, and standard test inputs.

5. [Laplace Transform Methods](05-laplace-transform-methods/index.md) - Laplace transform techniques for control analysis including s-domain representation and partial fractions.

6. [Poles, Zeros, and System Analysis](06-poles-zeros-system-analysis/index.md) - Relationship between pole-zero locations and system behavior.

7. [Physical System Modeling](07-physical-system-modeling/index.md) - Mathematical models for electrical, mechanical, and electromechanical systems.

8. [Linearization and Nonlinear Effects](08-linearization-nonlinear-effects/index.md) - Linearization techniques and common nonlinearities in control systems.

9. [Block Diagrams and Signal Flow](09-block-diagrams-signal-flow/index.md) - Graphical system representation including block diagram reduction and Mason's gain formula.

10. [Stability Analysis and Routh-Hurwitz](10-stability-routh-hurwitz/index.md) - Stability concepts, characteristic equations, and the Routh-Hurwitz criterion.

11. [Root Locus Analysis and Design](11-root-locus-analysis-design/index.md) - Root locus method for analyzing closed-loop pole movement with gain.

12. [Frequency Response and Bode Plots](12-frequency-response-bode-plots/index.md) - Frequency-domain analysis techniques including Bode plot construction and bandwidth.

13. [Nyquist Analysis and Stability Margins](13-nyquist-stability-margins/index.md) - Nyquist diagrams, stability criterion, and gain/phase margins.

14. [Steady-State Error Analysis](14-steady-state-error-analysis/index.md) - Steady-state accuracy, error constants, and system type classification.

15. [PID Control and Controller Tuning](15-pid-control-tuning/index.md) - Proportional, integral, and derivative control with tuning methods.

16. [Compensator Design and Performance](16-compensator-design-performance/index.md) - Lead/lag compensation and performance tradeoffs in control design.
